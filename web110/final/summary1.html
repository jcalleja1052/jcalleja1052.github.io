HTML Summary

When building a website, nowadays the main coding language nowadays to use is HTML, specifically HTML5. HTML stands for Hyper Text Markup Language and this is the current standard in the web development market for coding language to create a website with. Back then, PHP (Hypertext Preprocessor) was the main language to use in order to create websites. Overtime, the market and the field has drastically changed, with the addition of HTML5 being used in conjunction with JavaScript and CSS (Cascading Style Sheets).

With this, web browsers will take HTML documents and turn them into rendered pages that can be viewed. Without the HTML markup, a webpage will be a white background with black text. HTML is a very incredible tool for all web developers with how flexible it can be used since it can be used together with CSS and JavaScript, but this summary is mostly focusing on HTML.
