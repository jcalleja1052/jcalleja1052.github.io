For assignment 7, some new terms I learned from this assignment and the Wikiversity reading was the term 'Extranet'. I've heard of Intranet before but I never really thought of the opposite of Intranet, which is what Extranet basically is. Intranet is a private network inside of an enterprise that only allows company users, so the opposite of that would be a private network that gives access to people who are not just company workers, which is what Extranet is. Another term I learned more about from the session was more about DNS, or Domain Name System. I knew beforehand a bit about DNS and that it was used for domain names like google.com or github.com, but I actually never knew that it mainly done by converting IP addresses into domains, and that this system is used to make it a lot more simple to access websites without having to type in the full IP address of a site, and just simply using a domain name as the IP address and domain name will be connected together and the device connecting to the site will retrieve the IP address from the domain name in order to just connect to the site via the domain name and not by the IP address. One more new thing I learned from the session was a switch. I never really heard of what a switch does for networks and internet but can also be called bridge. The switch/bridge is a device that is supposed to connect to many network segments/devices and will filter out those segments/devices to send data to devices that are supposed to receive that data.
