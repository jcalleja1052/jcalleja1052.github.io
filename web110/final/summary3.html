JavaScript Summary

JavaScript is a programming language that has similar syntax to Java (hence the name), although they have completely different purposes. For instance, Java uses C-style syntax and is an object-oriented programming language with classes and objects. On the other hand, JavaScript also has a similar C-style syntax, but instead of being object-oriented, it is used for text and scripts with just names and elements similar to Java in general. It should not be said that JavaScript and Java are similar. They may have similar C syntax used in their language, but they used for completely different things. This language is also one of the main programming languages used with the World Wide Web, with HTML and CSS.

In order to implement JavaScript into an HTML document, we use the <script> tag in HTML. It can also be used in CSS, but it is highly recommended to add anything JavaScript related into an HTML file instead.
Example of using JavaScript in HTML:

<h1 id = "text"></h1>
<script>
document.getElementById("text").innerHTML = "Harper College";
</script>

What this code should basically display is:

In this example, right before we start working with the <script> tag, we create a new header tag of 'h1' with the ID of "text". This ID will be used to link the header with the JavaScript that is made in between the script tags. Explaining each different portion of this JavaScript code, 'document' is a variable that is using a method ('getElementbyId'). This method is exactly what you think it is, getting the element (the element in question being a string called "text"). It is then using this ID to create a string of text (innerHTML) that will be "Harper College".
In short, this is creating the text "Harper College" to be used by <h1> and this is done by linking the IDs with both the script and the header.

The reason why the JavaScript code is going after the html is because we want the HTML to load first before the script starts working, if the script loaded before the HTML then the script wouldn't work with the HTML.

Inside the style tag, with JavaScript you can also create what's known as functions that will usually end up using a lot of math involved in it, though this depends on what the function is created for. Simply think of a function as a group of code that can be called from another part of code that's not inside the code (however you can have the function call itself which is called recursion).

Here is an example of a basic function:

<h1 id = "display"></h1>
<script>
function finalExample {
let a, b, c;
b = 2000;
c = 23;
a = b + c;
return a;
}
</script>

What this code should basically display is:

Again before we start with the JavaScript, we make a header with an ID to link the header with the script we are making with document.getElementById().
Next, we now create the function called 'finalExample' for this example. The braces is used to keep all of the code in the function grouped and is required for the function to work.

In this code, we declare three different variables, a, b, and c using 'let'. The next lines is having b and c equal to two different numbers, and then letting a equal b and c added together. A more understandable version of this without the letters can be shown here describing each line of the code:

let a, b, c; - create 3 variables A, B, C.
b = 2000; - Have B equal to the number 2000
c = 23; - Have C equal to the number 23
a = b + c; - A = 2000 + 23, so A is 2023.
return a; - Have what A is equal to be the final result of this function (2023).

The 'return a;' in this function is a statement that ends the function and depending on what variable is being added to the return statement, that will be the variable sent to whenever this function is called.
