CSS Summary

When it comes using HTML, CSS, and JavaScript for website development, HTML is essentially the building blocks for a website as CSS is the styling portion of a website. However, since it is basically the barebones of a webpage, the other coding languages, CSS and JavaScript, come into play in order to bring more life to the webpage. This summary is focusing on CSS, so you can check the summary for JavaScript and HTML at the bottom of the page.

CSS, or Cascading Style Sheets, is a coding language that is focused on styling, which is mainly used with HTML for improving the look of the .html document. Many simple examples of what CSS can do include:

Adding style to a divider border in HTML with a box and colors

Changing the color of text and font (this font is Kanit Italic imported from Google fonts)

Having a .css file that can be used universally for .html documents (if they have the CSS stylesheet file linked with the .html file)

Improve the look of the content of a webpage in general and make it accessible for any size and device viewing the page

With CSS, there are multiple ways to add styling to your HTML document. The multiple ways to add styling to your HTML document is:

Internal
(using the <style> tag with HTML to add styling)

External
(similar to internal but instead of being in the HTML file it uses a separate file (.css) for the HTML to link with for styling)

Inline
(adding 'style="property: value;"' to a tag before ending the starting tag with >
Example of this is <p style = "text-align: center;">)

As said before, CSS can allow universal styling for HTML pages to be styled/formatted the same way as long as each HTML page is connected to the .css file. The way you can do this is with adding the link tag in the head section of the HTML document.
Example:

<link rel="stylesheet" href="style.css">

The element 'link' is used to define the relationship between the HTML document and an external file, in this case it is commonly use to link with CSS files or style sheets.

The attribute 'href' is referring to what destination the link element is pointing to, which is obviously style.css in this example since they are in the same folder.

If the external file (and this applies to 'href' being used in other situations) is located in a different folder that the current HTML file is not in, then this would require the user to type the destination towards the external file. For example, if my style sheet for this document was located in a folder called 'stylesheets', then I would have to type:

<link rel="stylesheet" href="stylesheets/style.css">

With all of these tools that CSS has to offer, it can really improve the appearance of an HTML page's content entirely. The amount of things CSS can do can create a stylistic website that can be unique and different to many other websites with HTML and CSS, however these two languages are lacking in one thing that JavaScript has: interactability.

HTML may be the building blocks of a webpage with headers and text and links, and CSS may be the stylistic aspect of the webpage with colors and customizability, but JavaScript is the one language out of the three that can allow the user browsing on the page to fully interact with the webpage.
